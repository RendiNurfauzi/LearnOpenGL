# LearnOpenGL

LearnOpenGL is a C++ project created for learning and experimenting with **OpenGL graphics programming**.
This project focuses on understanding the fundamentals of rendering, window management, and graphics pipelines.

---

## Features
- OpenGL-based rendering
- GLFW for window creation and input handling
- GLAD for OpenGL function loading
- Simple and clean project structure for learning purposes

---

## Requirements
- Visual Studio 2022
- C++17 or newer
- OpenGL 3.3+ compatible GPU

---

## Build Instructions
1. Clone this repository
2. Open `LearnOpenGL.sln` in Visual Studio
3. Select `x64` configuration
4. Build and run the project
